What's the problem?

The textbox must be empty when the "Go" button is clicked. Its content does not matter when anything else is clicked.

Detect it with Cornipickle

# The event is attached to its target through the # attribute "event". Its value is the event's type We say that I click on Go when ( There exists $b in $(button) such that ( ($b's text is "Go") And ($b's event is "mouseup") ) ). We say that the textbox is empty when ( For each $t in $(#textbox) ( $t's text is "" ) ). """ @name No empty box when Go @description When clicking on Go, the textbox must not be empty. This will return true when clicking on anything else. @severity Error """ If (I click on Go) Then (the textbox is empty).

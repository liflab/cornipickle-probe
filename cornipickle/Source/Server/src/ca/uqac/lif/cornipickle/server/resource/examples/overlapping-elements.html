What's the problem?

Two squares overlap (let's pretend they shouldn't). Try moving the squares around and re-evaluate the property.

Detect it with Cornipickle

# First condition We say that $x x-intersects $y when ( ($y's right is greater than $x's left) And ($x's right is greater than $y's left) ). # Second condition We say that $x y-intersects $y when ( ($y's top is greater than $x's top) And ($x's bottom is greater than $y's top) ). # Both must be true We say that $x and $y overlap when ( ($x x-intersects $y) And ($x y-intersects $y) ). # Define the negation to simplify the grammar We say that $x and $y don't overlap when ( Not ($x and $y overlap) ). """ @name Overlapping squares @description Coloured squares should not overlap in the page @severity Error """ For each $x in $(.square) ( For each $y in $(.square) ( ( $x's id equals $y's id ) # This condition is necessary, as x and y may be the same square Or ( $x and $y don't overlap ) ) ).

Procedural Equivalent

x_intersects = function(x, y) { return (y.clientLeft + y.width) > x.left && x.clientLeft + x.width > y.left; }; y_intersects = function(x, y) { return (y.clientTop + y.height) > x.top && x.clientTop + x.height > y.top; }; overlap = function(x, y) { return x_intersects(x, y) && y_intersects(x, y); }; for (var x in $(".square")) { for (var x in $(".square")) { if (!(x.id === y.id || !overlap(x, y))) console.log("Error"); } };
